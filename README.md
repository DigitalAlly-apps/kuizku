# KuizKu — Setup & Run Guide

## Install & Start

```bash
cd /home/miqdad/Documents/kuizku/kuizku
npm install
npm run dev
```

Buka browser ke: http://localhost:5173

## First Run

1. Klik **"Daftar sekarang"** untuk buat akun guru
2. Isi nama, email, password, dan mata pelajaran
3. Anda langsung masuk ke Dashboard
